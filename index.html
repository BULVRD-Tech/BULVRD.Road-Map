BULVRD Road Map

Product + Tech Jan 2018 - Future

This page serves as a high level outline of the road map for BULVRD. This is will be an ever living page, open to changes and updates. All historical aspects will be maintained via the dedicated repo on the BULVRD-Tech Github page. You will be able to explore deeper details and discussions on the BULVRD Wiki also found on our Github.

BULVRD Announced

Time will be spent harmonizing resources, spreading awareness, releasing and open sourcing tech demos, providing documentation, and finalizing structure.

January 2018

BULVRD Android Public App Beta Release

The initial Android public beta will include 3D mapping, Augmented Reality with ARCore, real time traffic, turn by turn navigation, ability to make 9+ road report types, and reward users points that will be able to be converted into BULVRD tokens after the full public release. The beta is currently available on Google Play App Store

June 7th 2018

BULVRD iOS App Public Beta Release

The initial iOS public beta will include 3D mapping, Augmented Reality with ARKit, real time traffic, turn by turn navigation, ability to make 9+ road report types, and reward users points that will be able to be converted into BULVRD tokens after the full public release.

Sept 2018

Token Sale

BULVRD will offer a token sale. Check our main site to see details on how to join. At the current time, this sale will account for up to 19% of the total supply. Please reference our Whitepaper for a more detailed distribution breakdown. Exact date and time TBA, will run for 60 days. Exact time will be announced via official group chats and social channels 48hrs before.

January 2019

BULVRD Android App Full Release

The initial Android full public release will include 3D mapping, Augmented Reality with ARCore, real time traffic, turn by turn navigation, ability to make 9+ road report types, and reward users points that will be able to be converted into BULVRD tokens. The app will be published on multiple Android app stores.

Post Token Sale

BULVRD iOS App Full Release

The initial iOS full public release will include 3D mapping, Augmented Reality with ARKit, real time traffic, turn by turn navigation, ability to make 9+ road report types, and reward users points that will be able to be converted into BULVRD tokens.

Post Token Sale

Machine Learning App Integrations

Machine Learning via Tensorflow will be rolled out to the public apps. This integration will allow for increased world scanning which will give users more options for earning BULVRD tokens, create automatic reporting systems, and improve data sets for cloud Machine Learning integrations.

2019

Machine Learning Data Integrations

Cloud Machine Learning via custom Tensorflow models will be leveraged on road reports to create predictive elements to improve overall user experince and mapping data points.

2019

BULVRD Advertising Platform : Phase 1

The BULVRD advertising platform will be open for the public in Phase 1. This platform will allow for buying location based ad placements displayed on the 3D maps, Augmented Reality views, and within location based feeds. Please explore the BUVLRD whitepaper for further details.

March 2019

Research

Research will continue throughout 2018. This includes tech improvements, new features, assessing and implementing community request, and expansions to new Augmented Reality platforms. As previously mentioned, this page will continue to change and update as our road map expands. Please visit the BULVRD-Tech projects page to explore what is being developed, and what future features are being explored.
