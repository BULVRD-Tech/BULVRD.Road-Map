BULVRD Road Map

Product + Tech Jan 2018 - Future

This page serves as a high level outline of the road map for BULVRD. This is will be an ever living page, open to changes and updates. All historical aspects will be maintained via the dedicated repo on the BULVRD-Tech Github page. You will be able to explore deeper details and discussions on the BULVRD Wiki also found on our Github.

BULVRD Announced

Time will be spent harmonizing resources, spreading awareness, releasing and open sourcing tech demos, providing documentation, and finalizing structure.

January 2018

BULVRD Android App Alpha Release

The initial Android private beta will include 3D mapping, Augmented Reality with ARCore, real time traffic, turn by turn navigation, ability to make 9+ road report types, and rewards a user acquire that be able to be converted into BULVRD tokens after the full public release. The beta is currently available on Google Play App Store

June 7th 2018

Begin ConsenSys Tachyon

BULVRD will begin incubation in the ConsenSys Ventures accelerator program, Tachyon.

Sept 2018

BULVRD iOS App Alpha Release

The initial iOS private beta will include 3D mapping, Augmented Reality with ARKit, real time traffic, turn by turn navigation, ability to make 9+ road report types, and rewards a user acquire that be able to be converted into BULVRD tokens after the full public release. The beta is currently available on TestFlight

Sept 2018

Complete ConsenSys Tachyon

BULVRD will complete incubation in the ConsenSys Ventures accelerator program, Tachyon.

Dec 2018

BULVRD Drive Public Beta Release

Initial launch of the BULVRD Drive app betas on Android, iOS, and Samsung Galaxy App Store.

Feb 2018

Phone Manufacture Partnership

BULVRD begins collaboration with a leading phone manufacture, to be announced at a later date.

Feb 2019

AWS Grant

BULVRD recieves a $100k grant from AWS to leverage their cloud computing ecosystem.

March 2019

Segment Partnership

BULVRD recieves grant from Segment to leverage their analytics platform.

March 2019

XYO Partnership

BULVRD announces a partnership with XYO Network to leverage their Proof of Location technology to add a new layer of validation to BULVRD traffic data sets.

April 2019

Keycard Partnership

BULVRD begins working with Status and their Keycard for secured Ethereum Wallet storage.

May 2019

PlanetM Grant

BULVRD will begin working with the State of Michigan in leveraging their mobility expertise to improve the BULVRD ecosystem.

May 2019

Main Ethereum Network Launch

The BLVD token will launch on the Main Ethereum network. All earnings will be available and supported across any systems that support ERC-20 tokens.

July 4th 2019

BLVD DEX Listings

BLVD will be listed on 2 Decentralized Exchanges, with direct integrations into the ecosystem for ease of use for users.

July 2019

BULVRD Hub Teaser

The BULVRD Hub teaser will be launched. Giving the community a glimpse of whats to come.

July 2019

BULVRD Collectible Map Styles

BULVRD will launch limited edition map styles to be used within the BULVRD Ecosystem. These will be available in limited quanities on OpenSea

July 2019

BULVRD Bounty Program

BULVRD will launch a bounty program to incentivise the community to grow.

July 2019

BULVRD Android App Full Release

The initial Android full public release will include 3D mapping, Augmented Reality with ARCore, real time traffic, turn by turn navigation, ability to make 9+ road report types, and acquire in-app earnings that will be able to be converted into BULVRD tokens. The app will be published on multiple Android app stores.

Late July 2019

BULVRD iOS App Full Release

The initial iOS full public release will include 3D mapping, Augmented Reality with ARKit, real time traffic, turn by turn navigation, ability to make 9+ road report types, and acquire in-app earnings that will be able to be converted into BULVRD tokens.

August 2019

ACM Testing Facility

BULVRD receives a grant from PlanetM. PlanetM is Michigan's mobility initiative representing the collective mobility efforts across the state. PlanetM connects you to Michigan’s mobility ecosystem — the people, places and resources dedicated to the evolution of transportation mobility.

August 2019

BULVRD Pro Subscription

BULVRD will leverage the ERC-1337 standard to offer a monthly Pro subscription to unlock further functionality across the BULVRD ecosystem.

August 2019

BULVRD Collectibles

BULVRD will launch limited edition digital collectibles to be used within the ecosystem. These will be available via OpenSea.

July 2019

BULVRD Hub Beta

BULVRD will release the initial web portal to allow access to expanded ecosystem features.

September 2019

BULVRD Web Store

BULVRD will launch an online web store allowing users to make purchases with their BLVD tokens.

September 2019

BULVRD Collectibles

BULVRD will begin to grant collectibles earned within the ecosystem for contributions made, or hidden unlocks made.

Octobere 2019

NFT Staking (Via Web)

Users will be able to stake NFTs from a web portal at this time. NFT staking will be available in-app at launch, but will only be staked for 7 days. Stakes done via web will never expire until the user cancels it or the NFT is removed from their wallet.

October 2019

Token Drops (Via Web)

You will be able to place and schedule token drops, providing a more automated system to the BULVRD location based token drops.

October 2019

BULVRD Advertising Platform : Phase 1

The BULVRD advertising platform will be open for the public in Phase 1. This platform will allow for buying location based ad placements displayed on the 3D maps, Augmented Reality views, and within location based feeds. Please explore the BUVLRD whitepaper for further details.

October 2019

BULVRD DAO

The BLVD token smart contract ownership will be transferred to a DAO at this time. Enabling token holders across the ecosystem to participate in governance.

Dec 2019

Machine Learning App Integrations

Machine Learning via Tensorflow will be rolled out to the public apps. This integration will allow for increased world scanning which will give users more options for earning BULVRD tokens, create automatic reporting systems, and improve data sets for cloud Machine Learning integrations.

2019

Machine Learning Data Integrations

Cloud Machine Learning via custom Tensorflow models will be leveraged on road reports to create predictive elements to improve overall user experince and mapping data points.

Late 2019

Geoblockchain Terminal

BULVRD will offer a Geoblockchain 'Bloomberg Terminal-esk' system to allow for direct data access and analysis across numerous location focused blockchain projects.

Late 2019

Research

Research will continue throughout 2019. This includes tech improvements, new features, assessing and implementing community request, and expansions to new Augmented Reality platforms. As previously mentioned, this page will continue to change and update as our road map expands. Please visit the BULVRD-Tech projects page to explore what is being developed, and what future features are being explored.
